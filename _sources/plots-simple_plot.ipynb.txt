    "======================================\n",
    "Plotting with the ``SimplePlot`` class\n",
    "======================================\n",
    "\n",
    "The classes in the :mod:`plots <aim2dat.plots>` sub-package can be used to analyse and plot data. Each class is dedicated to spcific data types and therefore implements different functions with the prefix ``import_*`` to import individual data sets. All data sets are stored in the internal attribute ``_data`` having the ``data_label`` as a key and the imported data as value.\n",
    "\n",
    "The ``plot``-function can then be used to create a plot of the corresponding data. Depending on the class several data sets can be plotted at the same time or in separate subplots. All plot classes share an extensive set of properties to change the look and customize the plot.\n",
    "\n",
    "As a starting point, we demonstrate a basic use case with the :class:`SimplePlot <aim2dat.plots.SimplePlot>` class which is designed as the most general and adjustable interface to plot-backends.\n",
    "\n",
    "Importing data sets\n",
    "===================\n",
    "\n",
